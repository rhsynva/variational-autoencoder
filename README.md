3. Reparameterization Trick via Lambda Layer: To bypass the bottleneck of non-differentiable stochastic sampling and allow backpropagation, I implemented the reparameterization trick manually. I created a custom sampling function that dynamically generates standard normal random noise (epsilon) and computes the stochastic latent vectors using the formula z = z_mean + exp(0.5 * z_log_var) * epsilon, wrapping the logic inside a Keras Lambda layer.
4. Generative Decoder Network: I built a matching decoder configuration to map the sampled latent parameters back into the original input space. The 2-dimensional vector z is passed through a dense layer of 64 units with ReLU activation, followed by a final reconstruction dense layer of 784 units utilizing a sigmoid activation function to output clear pixel-wise reconstruction probabilities.
5. Custom Variational Loss Layer: Instead of utilizing standard preset objective metrics, I programmed the standard Evidence Lower Bound (ELBO) loss manually. The cost function explicitly aggregates a feature-wise binary cross-entropy reconstruction loss with an analytical Kullback-Leibler (KL) divergence regularizer that enforces a standard normal Gaussian prior on the latent distribution. I wrapped this combination inside a custom Keras Layer (VAE_Loss_Layer) using self.add_loss to ensure seamless internal loss collection during training loops.
6. Compilation, Training, and Generative Plotting: I compiled the full generative architecture with the Adam optimizer and fit the network to the training data over 20 epochs with a batch size of 128, tracking validation performance.
